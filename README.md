# 数当てゲーム (Number Guessing Game)

1から100までのランダムな数字を当てる、シンプルなWebブラウザゲームです。
HTML, CSS, JavaScriptの学習のために作成しました。

## 🎮 遊び方
1. 入力欄に **1〜100** の数字を入力します。
2. 「予想する」ボタンを押します。
3. ヒント（もっと大きい/小さい）を元に、**5回以内** に正解を目指してください！
